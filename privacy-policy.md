# Privacy Policy

**Last updated:** December 12, 2025

Cidade Quantica Monitor ("the application") respects your privacy. This policy explains how (or rather, **how little**) we handle data.

## 1. No Personal Data Collection
The application **does not collect, store, or transmit** any personal data, including:
- User IDs
- Message content beyond temporary local analysis
- IP addresses
- Server information

## 2. Local Processing Only
All message analysis occurs **locally on the owner’s machine**. No data is sent to external servers, cloud services, or databases.

## 3. Public Messages Only
The bot only accesses messages in **public channels** where it has been granted read permission. It never accesses DMs or private servers without explicit invite.

## 4. No Tracking
There is no analytics, cookies, or tracking of any kind.

## 5. Compliance
This application complies with Discord’s Privacy Policy and is designed to minimize data exposure.
